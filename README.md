# ohce-kata
Solution for Guidesmiths christmas kata by Giorgio &amp; Mario.

Install globally
```
npm i -g ohce-kata
```

After installing, you will be able to interact with the app like this:
```
$ ohce-kata Pedro
> ¡Buenos días Pedro!
$ hola
> aloh
$ oto
> oto
> ¡Bonita palabra!
$ stop
> pots
$ Stop!
> Adios Pedro
```
